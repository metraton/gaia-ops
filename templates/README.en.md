## ðŸ“‹ Available Templates

### `CLAUDE.template.md` (~200 lines)
Main template for generating orchestrator file.

**Placeholders:**
```
{{GITOPS_PATH}}        - Path to GitOps directory
{{TERRAFORM_PATH}}     - Path to Terraform directory
{{GCP_PROJECT_ID}}     - GCP project ID
{{GCP_REGION}}         - Primary GCP region
{{CLUSTER_NAME}}       - Cluster name
```

**Generated by:** `bin/gaia-init.js`  
**Output file:** `./CLAUDE.md`

## ðŸ”§ Technical Details

### Placeholder System

Placeholders use `{{NAME}}` syntax:

```javascript
const template = fs.readFileSync('templates/CLAUDE.template.md', 'utf8');
const rendered = template.replace(/\{\{GITOPS_PATH\}\}/g, gitopsPath);
```

### Versioning

Templates follow package versioning:
- **MAJOR:** Breaking changes in placeholder structure
- **MINOR:** New optional placeholders
- **PATCH:** Template corrections, text improvements
